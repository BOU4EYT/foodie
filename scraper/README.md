# Grocery Price Scraper

Targeted scraper for Walmart grocery category pages, scoped to the household's local store/zip.

## Target categories (v1)

- Meat & Seafood
- Fresh Produce
- Bakery & Bread
- Dairy & Eggs
- Frozen
- Pantry
- Snacks

## Notes

- Uses a headless browser (Playwright) since category pages are dynamic/paginated
- Store/zip must be set in the scraping session for prices to reflect the correct local store
- Refresh cadence: every 12-24 hours
- Personal/household use only — not intended to scale to other users without revisiting Walmart's ToS
